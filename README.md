# GIANT MOON v0.3

A lightweight Three.js/Vite prototype.

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Controls
- WASD move
- Mouse drag look
- Space jump
- Escape pause
- Mobile joystick + jump button

## Notes
Invert Y is default. Mundane achievements pay the most XP.
